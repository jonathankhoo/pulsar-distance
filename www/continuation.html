Module 1 continuation (under development)

Here is a list of projects that can be undertaken with module 1

  • Try and determine which is the most distant pulsar in the PULSE@Parkes sample.
  • See how the dispersion measure changes for different observations of the same pulsar. Why does the dispersion measure change? Make a plot of the dispersion measure as a function of time. Does it change randomly or systematically?
  • You can use professional tools to obtain more information about the pulsars. Use the ATNF pulsar catalogue to determine the pulsar's position, spin period and dispersion measure. You can use these positions in SkyView to obtain images of the sky near the pulsar using different telescopes.
