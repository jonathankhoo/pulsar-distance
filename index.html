PULSE@Parkes: Module One

Welcome to PULSE@Parkes Module One. The PULSE@Parkes project allows high-school students to carry out real-time observations of pulsars using the Parkes 64-m radio telescope situated in New South Wales, Australia. In this module, you will learn some basic information about pulsars and radio astronomy. The aim of the module is to measure the distance to a pulsar using recent PULSE@Parkes observations.
